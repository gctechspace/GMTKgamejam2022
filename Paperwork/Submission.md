# Submission Rules

* Upload game to [itch.io](https://itch.io/game/new)

## Upload Requirements
- Title
- Project URL
- Short Description
- Classification
- Kind of Project
- Release Status
- Pricing
- Uploads
    - 1GB File Size Limit
- Details
    - Description - HTML Game Page
- Genre
- Tags
- App Store Links
- Custom Noun
- Upload Cover Image
